# ACAI — Archive of Contemporary Art-events In Korea

한국 미술 이벤트(전시·프로그램·오픈콜) 통합 디지털 아카이브. MMCA·SeMA·ACC 공식 사이트와 아트누리에서 수집한 데이터를 정규화한 정적 웹 애플리케이션입니다.

- Live: https://felixnam.github.io/acai/
- 규모: art events 13,500+ · works 17,900+ · participants 9,800+ · Getty AAT 연계 키워드 체계
- 구조: vanilla ES modules SPA (해시 라우터), `data/`의 정적 JSON을 클라이언트에서 로드
- 데이터셋 논문: JOHD 제출 준비 중
